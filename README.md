<h1 align="center">Sistema de Votaciones - Centro de Estudiantes</h1>
<p align="center">Aplicación desarrollada en Python con CustomTkinter para gestionar de forma digital las elecciones del centro de estudiantes, evitando el uso de urnas físicas y agilizando el conteo de votos.</p>

## 📌 **Objetivos principales:**  
- Digitalizar el proceso de votación escolar.  
- Evitar errores y duplicación de votos.  
- Asegurar que cada estudiante vote una sola vez mediante correo institucional.  
- Ofrecer una experiencia clara y simple para los usuarios.  

## ⚙️ **Funcionalidades principales:**  
- Pantalla de inicio con acceso al sistema.  
- Formulario de registro con validación de correo institucional.  
- Sistema de votación con listas y voto en blanco.  
- Prevención de votos duplicados.  
- Pantalla de confirmación con mensaje de éxito.  
- Registro automático de votos en base de datos.
